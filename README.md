## How to Deploy
1. Clone the repo:
   ```bash
   git clone https://github.com/USERNAME/devops-resume-terraform.git

2. Navigate to the project directory:
   ```bash
   cd devops-resume-terraform

3. Initialize Terraform:
   ```bash
   terraform init
4. Apply the configuration:
   ```bash
   terraform apply -auto-approve
